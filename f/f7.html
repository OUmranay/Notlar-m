.fcl Dosyasının Yapılandırılması

Modelin tamamı bir FUNCTION_BLOCK içine yazılır. CamasirMakinesiModel.fcl adında bir dosya oluşturulur.

Adım 1: Girdi Değişkenleri (VAR_INPUT)

Modelin hangi girdileri alacağı VAR_INPUT ve END_VAR blokları arasına yazılır. Tür olarak REAL (ondalıklı sayı) kullanılır.

VAR_INPUT
    Kirlilik_Miktari : REAL;
    Camasir_Turu : REAL;
END_VAR

Adım 2: Çıktı Değişkenleri (VAR_OUTPUT)

Modelin hangi çıktıyı üreteceği VAR_OUTPUT ve END_VAR blokları arasına yazılır.

VAR_OUTPUT
    Yikama_Suresi : REAL;
END_VAR

Adım 3: Bulanıklaştırma (FUZZIFY)

Girdilerin net sayısal değerlerini (örn: kirlilik 70) bulanık dilsel ifadelere (örn: Az, Orta, Fazla) dönüştüren üyelik fonksiyonları burada tanımlanır. TERM ile terim, trap ile yamuk, triang veya koordinatlarla üçgen fonksiyon tanımlanır.

Girdi 1: Kirlilik Miktarı
FUZZIFY Kirlilik_Miktari
    TERM Az := (0, 1) (50, 0) ;
    TERM Orta := (0, 0) (50, 1) (90, 0) ;
    TERM Fazla := trap (60, 0) (80, 1) (100, 1) (120, 0) ; 
END_FUZZIFY
Girdi 2: Çamaşır Türü
FUZZIFY Camasir_Turu
    TERM Narin := (0, 1) (50, 0) ;
    TERM Karisik := (0, 0) (50, 1) (90, 0) ;
    TERM Beyaz := trap (60, 0) (80, 1V) (100, 1) (120, 0) ;
END_FUZZIFY

Adım 4: Durulama (DEFUZZIFY)

Çıktı değişkeninin bulanık kümeleri ve bu bulanık sonucu net bir sayıya dönüştürme metodu burada tanımlanır. Metot olarak COG (Center of Gravity) seçilmiştir. DEFAULT ise hiçbir kural çalışmazsa atanacak varsayılan değerdir.

DEFUZZIFY Yikama_Suresi
    TERM Cok_Kisa := trap (0, 1) (0, 1) (15, 1) (30, 0) ;
    TERM Kisa := triang (20, 0) (40, 1) (60, 0) ;
    TERM Normal := triang (40, 0) (60, 1) (90, 0) ;
    TERM Uzun := triang (60, 0) (90, 1) (100, 0) ;
    TERM Cok_Uzun := trap (90, 0) (100, 1) (145, 1) (145, 1) ;

    METHOD : COG;
    DEFAULT := 90;
END_DEFUZZIFY

Adım 5: Kural Bloğu (RULEBLOCK)

Girdiler ve çıktılar arasındaki mantıksal bağlantıların (kuralların) yazıldığı yerdir. AND operatörü için MIN, toplama (Aggregation) için MAX metodu seçilerek bir Min-Max çıkarım sistemi oluşturulmuştur.

Operatör Tanımları
RULEBLOCK Rule_Block_1
    AND : MIN;
    ACT : MIN;
    ACCU : MAX;
    ...
END_RULEBLOCK
Kural Matrisi (Toplam 9 Kural)
IF Kirlilik AND Çamaşır Türü THEN Yıkama Süresi
Az Narin Cok_Kisa (Kural 1)
Az Karisik Kisa (Kural 2)
Az Beyaz Normal (Kural 3)
Orta Narin Kisa (Kural 4)
Orta Karisik Normal (Kural 5)
Orta Beyaz Uzun (Kural 6)
Fazla Narin Normal (Kural 7)
Fazla Karisik Uzun (Kural 8)
Fazla Beyaz Cok_Uzun (Kural 9)

CamasirMakinesi.java

Bu sınıf modeli yükler, girdileri ayarlar, çalıştırır ve sonucu döndürür.

  • FIS (Fuzzy Inference System): Modelin kendisini temsil eden ana nesne.
  • FIS.load(): .fcl dosyasını yükler.
  • fis.setVariable(): Girdi değişkenlerine net değerleri atar.
  • fis.evaluate(): Tüm bulanık mantık sürecini (bulanıklaştırma, çıkarım, durulama) çalıştırır.
  • fis.getVariable().getValue(): Çıktı değişkeninin net (durulaştırılmış) değerini alır.
// Örnek Java Kodu Yapısı
public class CamasirMakinesi {
    private FIS fis;
    private double yikamaSuresi;

    public CamasirMakinesi(double kirlilik, double tur) {
        String fclDosyasi = "CamasirMakinesiModel.fcl";
        fis = FIS.load(fclDosyasi, true);

        fis.setVariable("Kirlilik_Miktari", kirlilik);
        fis.setVariable("Camasir_Turu", tur);
        fis.evaluate();

        yikamaSuresi = fis.getVariable("Yikama_Suresi").getValue();
    }
    
    public String toString() {
        return "Yıkama Süresi: " + Math.round(yikamaSuresi) + " dk";
    }
}

Kullanıcıdan Scanner ile girdileri alır ve CamasirMakinesi nesnesini oluşturur.

// Main metodu
public static void main(String[] args) {

    // Scanner ile camasirTuru (örn: 30) al

        70, 30
    );
    
    System.out.println(makine);
}
