HAFTA 1: Temel Kavramlar, Tanım ve Felsefe

Bulanık Mantık Nedir?

Klasik mantığı genişleten, belirsizlik ve kesin olmama durumlarını ele alan matematiksel bir çerçevedir.

Ayrıntılar için üzerine gelin.

Kısmi Üyelik

Bulanık Mantık, tam üyelik (0 veya 1) yerine kısmi üyeliği (0 ile 1 arası) kabul eder. Böylece "sıcak", "uzun" gibi dilsel ifadelerle çalışabilir.

  • Temel Kavram: Belirsizliğin modellenmesi.

Kurucu ve Felsefe

1965 yılında Prof. Lotfi Zadeh tarafından tanıtılmıştır.

Klasik Mantık ve Bulanık Mantık Karşılaştırması

Klasik Kümeler

Üyelik yalnızca iki değeri alabilir: 1 (tam üye) veya 0 (üye değil). Sınırlar keskindir.

Örneğin, bir kişi ya tam olarak "Uzun" kümesine aittir ya da değildir. 1.79m değilse, 1.80m altındaki herkes aynı muameleyi görür.

μA(x)∈{0,1}

Bulanık Kümeler

Üyelik 0 ile 1 arasında herhangi bir değer alabilir. Bu, bir nesnenin bir kümeye kısmen ait olabileceği anlamına gelir.

Örneğin, bir kişi 1.79m ise, "Uzun" kümesine 0.8 üyelik derecesiyle ait olabilir. Sınırlar yumuşak ve geçişkendir.

μA(x)∈[0,1]

Olasılık ve Olabilirlik Karşılaştırması

Bir olayın gelecekte gerçekleşme ihtimali veya tekrar sıklığı ile ilgilidir.
Örnek: "Şişedeki sıvı %50 ihtimalle saf sudur." (Olay ya doğrudur ya yanlıştır, biz emin değiliz.)

Matematiksel Prensip (Alan)

Tüm olası sonuçların toplam ihtimali 1'dir: ∫−∞∞f(x)dx=1

Bir ifadenin ne derecede doğru olduğu veya bir kümeye ait olma derecesi ile ilgilidir. (Üyelik Fonksiyonu)
Örnek: "Şişedeki sıvı %50 oranında saf sudur." (Kesin bir bilgi, ifadenin kendisi kısmen doğru.)

Matematiksel Prensip (Maksimum)

Bir aralık için olabilirlik, o bölgedeki maksimum üyelik derecesidir: max{μ(x)|x∈I}

Bulanık Mantık Sistemlerinin 4 Temel Kavramı

1. Bulanık Kümeler

Geçişken sınırlara sahip, kısmi üyelik sunan kümeler.

2. Dilsel Değişkenler ve Terimler

Hız (Çok Yavaş, Yavaş, Hızlı gibi) veya Sıcaklık (Soğuk, Ilık, Sıcak gibi) gibi insana özgü kavramlar.

3. Üyelik Fonksiyonları

Giriş değerinin (örn: 30°C) dilsel terime (örn: Sıcak) ne kadar ait olduğunu gösteren grafiksel temsiller (μA(x)).

4. Bulanık Kurallar

"Eğer Hız Yüksek ise, Fren Basıncı Orta olmalıdır." şeklindeki karar mekanizmaları.

Geleneksel Programlama vs. Bulanık Mantık


// Geleneksel Yaklaşım (Keskin Sınırlar)
// Hız: 0.24 = Çok Yavaş, 0.25 = Yavaş (Ani Sıçrama)

double hiz; // 0.0 - 1.0 arası değer
scanf(hiz);

if (hiz >= 0.0 && hiz < 0.25) {
    // Tamamen 'Çok Yavaş'
} else if (hiz >= 0.25 && hiz < 0.5) {
    // Tamamen 'Yavaş'
} else {
    // ...
}

// Bulanık Yaklaşım (Kısmi Üyelik)
// Hız 0.24 olabilir:
//   'Çok Yavaş' kümesine üyelik: 0.96
//   'Yavaş' kümesine üyelik: 0.04
// Karar, bu derecelere göre alınır. Sınırlar esnektir.
